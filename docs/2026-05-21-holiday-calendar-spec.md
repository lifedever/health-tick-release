# 国家节假日日历查看 — 产品规格说明

| 项目 | 说明 |

| 所属产品 | HealthTick（macOS 菜单栏久坐提醒） |
| 版本上下文 | v1.6.13+ |
| 文档类型 | 产品规格（Spec） |

---

## 1. 背景与目标

HealthTick 已支持从第三方接口同步**中国大陆**国务院公布的放假与调休安排，用于判断某日是否计入「工作日」（影响计时器、休息时段等逻辑）。此前用户只能看到「上次同步时间」，无法直观核对同步结果是否正确。

**本次变更目标：**

- 同步完成后，用户能**立即**以月历形式查看节假日安排；
- 支持随时打开日历，核对放假、调休上班与普通工作日/休息日；
- 展示节假日名称（如「国庆节」），降低理解成本。

---

## 2. 用户场景

| 编号 | 场景 | 期望结果 |
|------|------|----------|
| US-1 | 用户首次开启「同步国家节假日」且无本地数据 | 自动触发同步；成功后弹出日历 |
| US-2 | 用户点击「立即同步」更新数据 | 同步成功后自动弹出日历；显示最新安排 |
| US-3 | 用户已同步过，想确认某月调休 | 点击「查看节假日日历」，切换月份并点击日期查看详情 |
| US-4 | 用户未同步或数据为空时打开日历 | 提示「请先同步节假日数据」 |
| US-5 | 用户查看某日是否会计入工作日 | 详情区显示「计入工作日 / 不计入工作日」 |

---

## 3. 功能范围

### 3.1 包含（In Scope）

1. **入口**
   - 设置 → **应用** 标签 →「工作日」区块内；
   - 仅在「同步国家节假日」开关为**开启**时显示相关控件。

2. **操作**
   - 「立即同步」：行为与既有逻辑一致，成功后**自动打开**日历 Sheet；
   - 「查看节假日日历」：有同步数据时可点；无数据时禁用。

3. **日历展示**
   - 月视图（7 列 × 若干行），周日为每周第一天（与中国常见月历一致）；
   - 上一月 / 下一月切换；
   - 年份下拉（数据中出现的年份；无数据时默认可浏览当前年及次年）；
   - 点击日期显示详情面板。

4. **日期分类与图例**（四种）

   | 类型 | 含义 | 是否计入工作日 |
   |------|------|----------------|
   | 放假 | 国务院安排中的法定放假 | 否 |
   | 调休上班 | 国务院安排中的补班 | 是 |
   | 工作日 | 未在国务院表中，且符合用户「默认工作日」星期设置 | 是 |
   | 休息日 | 未在国务院表中，且不在默认工作日星期中 | 否 |

5. **日期详情**
   - 完整日期（本地化格式）；
   - 类型标签；
   - 是否计入工作日；
   - 节假日名称（若 API 提供且已同步）；
   - 来源说明：「国务院安排」或「按星期设置」。

6. **国际化**
   - 中文 / 英文文案（跟随应用语言设置）。

7. **数据**
   - 同步时除工作日覆盖表外，额外保存节假日**名称**映射。

### 3.2 不包含（Out of Scope）

- 编辑、手动增删节假日；
- 非中国大陆节假日数据源；
- 独立窗口（采用 Sheet 即可）；
- 导出日历或打印；
- 与系统「日历.app」集成。

---

## 4. 交互与界面要求

### 4.1 布局

- Sheet 宽度约 **420pt**；
- 顶部：标题「国家节假日安排」+ 关闭按钮；
- 主体：月导航 → 星期表头 → 日期网格 → 图例 → 选中日详情。

### 4.2 视觉编码

| 类型 | 建议色语义 |
|------|------------|
| 放假 | 橙色强调 |
| 调休上班 | 蓝色强调 |
| 默认工作日 | 浅绿背景 |
| 默认休息日 | 浅灰背景 |
| 选中日期 | 强调色描边 |

国务院表内日期在格子上使用**加粗日期数字**；有名称时显示小圆点提示。

### 4.3 状态

| 状态 | 表现 |
|------|------|
| 同步中 | 「立即同步」显示 Progress +「同步中…」，按钮禁用 |
| 同步失败 | 红色错误文案（网络 / API / URL） |
| 无数据 | 日历 Sheet 内空状态插画 + 提示文案 |
| 有数据 | 完整月历与图例 |

## 5. 业务规则

1. **工作日判定**（与既有逻辑一致，日历仅可视化）  
   - 开启节假日同步且该日在 `holiday_calendar` 中有记录：  
     - 值为「非工作日覆盖」→ 放假；  
     - 值为「工作日覆盖」→ 调休上班。  
   - 无记录：回退到用户设置的周一至周日开关。

2. **时区**  
   - 所有日期键与月历展示以 **Asia/Shanghai** 为准。

3. **同步范围**  
   - 每次同步拉取**当前自然年**与**下一年**数据并合并。

4. **自动打开日历**  
   - 仅在「立即同步」**成功完成**后触发；
   - 用户仅点击「查看节假日日历」时不重复同步。

## 6. 验收标准

- [ ] 开启节假日同步且同步成功后，日历 Sheet 自动弹出。
- [ ] 「查看节假日日历」在无数据时禁用，有数据时可打开。
- [ ] 月历中放假、调休、默认工作日/休息日四类视觉可区分。
- [ ] 点击国务院安排内的日期，详情显示名称（若 API 有）及「国务院安排」。
- [ ] 点击非表内日期，详情显示「按星期设置」且类型与下方星期按钮一致。
- [ ] 详情中「计入工作日」与计时器实际 `isWorkDay` 行为一致。
- [ ] 中英文界面文案正确。
- [ ] 重启应用后，日历仍可打开且数据与数据库一致。
