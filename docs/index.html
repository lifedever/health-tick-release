Features
专为久坐族设计
轻量常驻菜单栏，不打扰工作，但绝不让你忘记休息。
🕐

智能计时

自定义工作时长和休息时长，菜单栏实时显示倒计时进度。

💪

强制休息

休息时弹出提醒窗口，支持悬浮窗、全屏强制等多种模式。

🔍

操作检测

休息期间检测到键鼠操作自动暂停倒计时，确保真正休息。

🎯

每日目标

设定每天休息次数目标，7 天像素图直观追踪完成进度。

🔥

连续打卡

记录连续达标天数，断签不惩罚未使用的日子，鼓励坚持。

🏆

徽章激励

18 枚徽章等你解锁，连续打卡 + 累计打卡双线成就体系。

How it works
四步开始健康节奏
1

工作计时

启动后自动开始倒计时，菜单栏图标显示进度。

2

休息提醒

工作时长结束，弹出提醒窗口，可设置是否需要手动确认。

3

强制休息

进入休息倒计时，检测到操作会自动暂停，直到你真正休息。

4

继续工作

休息结束自动开始下一轮，打卡记录 +1，向徽章迈进。

Install
两种方式，即刻开始

🍺 Homebrew

一行命令，自动更新

# 安装
brew tap lifedever/tap
brew install --cask health-tick

# 更新
brew upgrade --cask health-tick

📦 手动下载

下载 DMG，拖拽安装

1

前往 Releases 页面下载 .dmg 文件

2

打开 DMG，将 HealthTick 拖入 Applications 文件夹

3

启动应用，完成引导设置，开始健康打卡

💡 首次打开可能提示"无法验证开发者"，请前往系统设置 → 隐私与安全性点击"仍要打开"。
